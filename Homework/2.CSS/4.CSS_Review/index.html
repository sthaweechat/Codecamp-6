9. grid และ flex แตกต่างกันอย่างไร Flex ใช้เพื่อจัดการ layout 1D ส่วน Grid ใช้เพื่อจัดการ layout 2D
Flex เราสามารถจัดวางของต่างๆภายใน layout 1 แถว อาจจะเป็นแนวตั้งหรือแนวนอน
Grid เราสามารถจัดของไว้ตรงไหนก็ได้ใน grid layout เช่นตัวอย่าง เราบอก grid ว่าให้ layout ของ header เต็ม row บรรทัดต่อมาให้ menu เป็น 2 columns content เป็น 8 columns อะไรประมาณนั้น
ใช้ flex เมื่อ layout มีลักษณะการวางของต่างๆเป็นแถวเดียว ไม่แนวตั้งก็นอน (1D)
ใช้ grid เมื่อ items ด้านในเรามีการจัดเรียงตัวที่ไม่ได้มีแค่แถวเดียว แต่มีลักษณะเป็น column กับ row (2D)
10. justify and align แตกต่างกันอย่างไร justify เป็นการกระจาย ส่วน align เป็นการจัดการเช่น ซ้าย ขวา เป็นต้น
